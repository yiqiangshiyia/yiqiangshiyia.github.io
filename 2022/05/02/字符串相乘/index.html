字符串相乘

LeetCode 43：字符串相乘

Note

对每次得到的字符串进行累加，可以使用「415. 字符串相加」的做法。

题目大意

给定两个以字符串形式表示的非负整数 num1 和 num2，返回 num1 和 num2 的乘积，它们的乘积也表示为字符串形式。

注意：不能使用任何内置的 BigInteger 库或直接将输入转换为整数。

题解

方法1

如果 num1 和 num2 之一是 0，则直接将 0 作为结果返回即可。

如果 num1 和 num2 都不是 0，则可以通过模拟「竖式乘法」的方法计算乘积。从右往左遍历乘数，将乘数的每一位与被乘数相乘得到对应的结果，再将每次得到的结果累加。这道题中，被乘数是 num1，乘数是 num2。

需要注意的是，num2 除了最低位以外，其余的每一位的运算结果都需要补 0。

AC代码：

class Solution {
public:
    string multiply(string num1, string num2) {
        if(num1 == "0" || num2 == "0")  return "0";
        string ans = "0";
        int m = num1.size(),n = num2.size();
        for(int i=n-1; i>=0; i--)
        {
            string cur;
            int add = 0;
            //末尾补0
            for(int j=n-1; j>i; j--)
            {
                ans.push_back(0);
            }
            int y = num2[i]-'0';
            for(int j=m-1; j>=0; j--)
            {
                int x = num1[j]-'0';
                int sum = x*y+add;
                cur.push_back('0'+sum%10);
                add = sum/10;
            }
            while(add != 0)
            {
                cur.push_back(add%10);
                add = add/10;
            }
            reverse(cur.begin(),cur.end());  //字符串翻转
            ans = addStrings(ans,cur);
        }
        return ans;
    }
    //字符串相加 LeetCode 415
    string addStrings(string &num1,string &num2)
    {
        int i = num1.size()-1,j = num2.size()-1,add = 0;
        string ans;
        while(i>=0 || j>=0 || add != 0)
        {
            int x,y;
            if(num1[i] >= 0)
            {
                x = num1[i] - '0';
            }else{
                x = 0;
            }
            if(num2[j] >= 0)
            {
                y = num2[j] - '0';
            }else{
                y = 0;
            }
            int sum = x+y+add;
            ans.push_back('0'+sum%10);
            add = sum/10;
            i--;
            j--;
        }
        reverse(ans.begin(),ans.end());
        return ans;
    }
};

方法2

方法1的时间和空间复杂度较高，方法一的做法是从右往左遍历乘数，将乘数的每一位与被乘数相乘得到对应的结果，再将每次得到的结果累加，整个过程中涉及到较多字符串相加的操作。如果使用数组代替字符串存储结果，则可以减少对字符串的操作。
