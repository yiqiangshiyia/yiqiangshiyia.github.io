一题四解：排序 集合 哈希表 位运算

LeetCode 136：只出现一次的数字

Note

这道题 虽然是一道简单题，但是学到了很多东西。意识到大一学的C++其实有很多东西都没有学到，比如哈希表和集合的很多用法都不够了解。  

题目大意

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。  

题解

方法1：判断排序数组前后元素是否相等

AC代码：

class Solution {
public:
    int singleNumber(vector<int>& nums) {
        sort(nums.begin(),nums.end());
        int n = nums.size();
        if(n == 1)  return nums[0];
        if(n>=2 && nums[0] !=  nums[1])  return nums[0];
        if(n>=2 && nums[n-1] != nums[n-2])  return nums[n-1];
        for(int i=1; i<n-1; i++)
        {
            if(nums[i] != nums[i-1] && nums[i] != nums[i+1])
            {
                return nums[i];
            }
        }
        return 0;
    }
};

方法2：集合

set集合只能使用迭代器遍历，不能使用下标遍历。所以只能返回set1.begin()，而不能返回下标。 auto ans = set1.begin(); //这里只能用auto 而不能用
int return *ans; //返回的必须是指针类型 因为set1.begin()是迭代器类型，所以 ans只能用auto定义，而不能用int类型定义。因此返回的值也必须是 指针类型的。  

AC代码：

class Solution {
public:
    int singleNumber(vector<int>& nums) {
        unordered_set<int> set1;
        for(int n : nums)
        {
            if(set1.count(n))
            {
                set1.erase(n);
            }
            else
            {
                set1.insert(n);
            }
        }
        auto ans = set1.begin();  //这里只能用auto 而不能用int
        return *ans;  //返回的必须是指针类型
    }
};

方法3：哈希表

哈希表增加键值对的函数：map1.insert(make_pair(n,1));
哈希表使用迭代器遍历时： i->first：表示键 i->second：表示值  

AC代码：

class Solution {
public:
    int singleNumber(vector<int>& nums) {
        map<int,int> map1;
        for(int n : nums)
        {
            if(map1.find(n) == map1.end())
            {
                map1.insert(make_pair(n,1));  //如果字典中不存在该key，则新增一个键值对(n,1)
            }
            else
            {
                map1[n]++;  //如果字典中存在该key，则将值加1
            }
        }
        for(auto i=map1.begin(); i!=map1.end(); i++)
        {
            if(i->second == 1)
            {
                return i->first;
            }
        }
        return 0;
    }
};

Note：相同类型的哈希表题目：LeetCode 169：多数元素  

方法4：位运算

这个方法太精妙了！  

AC代码：

class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ret = 0;
        for (auto e: nums) ret ^= e;
        return ret;
    }
};
