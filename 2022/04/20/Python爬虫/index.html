Python爬虫

Python爬虫学习

学习爬虫前需要安装的标准库

安装如下：

pip install requests

pip install re

pip install bs4

爬虫的四个主要步骤：

1.明确目标

2.爬(将网站的所有内容给爬下来)

3.取(将自己想要的数据取下来)

4.保存数据

接下来的所有代码操作都是在执行爬虫的四个主要步骤

import requests #爬取html的标准库函数

ret=requests.get(‘http://www.qiushibaike.com/') #ret返回的是网页请求成功的状态码

result=ret.content.decode() #通过这段代码能够把网页的中文源代码返回给我们

print(result) #result是一个字符串

通过请求一个url，来得到前端的数据，然后从前端的数据里面获取我们想要的内容

import re #python的一个模块 用来将自己想要的数据取下来

re模块是针对字符串这种数据格式来提取的一种方法

具体代码：

import re
import requests  #爬取html的标准库函数
headers={
    "User-Urgent":"Mozilla/5.0 (Windows NT 10.0; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/90.0.4430.93 Safari/537.36"
}  #防反爬 模拟人去访问网站 
ret=requests.get('https://www.xbiquge.la/paihangbang/')   #ret返回的是网页请求成功的状态码
result=ret.content.decode()   #通过这段代码能够把网页的中文源代码返回给我们
#result是一个字符串
novel = '<li>\d<a href=".+">(.*?)<\/a><\/li>'
content=re.findall(novel,result)
#re.S代表可以匹配到换行符
#print(content)
with open('笔趣阁爬虫数据.txt','w',encoding='utf-8') as f:
    for i in content:
         s = i+'\n'
         f.write(s)

一个完整的爬虫实例（BeautifulSoup模块解析）

import requests  
from bs4 import BeautifulSoup
headers={
    "User-Urgent":"Mozilla/5.0 (Windows NT 10.0; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/90.0.4430.93 Safari/537.36"
}
ret=requests.get('https://www.xbiquge.la/paihangbang/') 
result=ret.content.decode() 
#print(type(result)) 
soup = BeautifulSoup(result,'html.parser')
#print(type(soup))
content = soup.find_all('li')
for i in content:
    print(i.text)

写给小吕的详细解释版

import re  #re为python的一个模块 用于解析网页源代码
#同样可以实现该功能的还有bs4 lxml等
import requests  #爬取html的标准库函数

headers={
    "User-Urgent":"Mozilla/5.0 (Windows NT 10.0; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/90.0.4430.93 Safari/537.36"
}  #防反爬 模拟人去访问网站 

ret=requests.get('https://www.xbiquge.la/paihangbang/')   #ret返回的是网页请求成功的状态码
#requests.get()可以得到该网站的代码

result=ret.content.decode()   #对代码进行解析 通过解析能够把网页的中文源代码返回给我们
#得到中文源代码之后只需要提取我们所需要的的数据即可
#result是一个字符串

novel = '<li>\d<a href=".+">(.*?)<\/a><\/li>'
#使用正则表达式去匹配代码中的关键字 小括号()前后代表所要去匹配代码的标签
#(.*?)代表所要提取的内容
#这个地方你在展示的时候如果说不明白的话 你可以这么说：
#正则表达式是爬虫的关键 用来提取所爬代码中我们所需要的数据 如果大家想要更深入的了解的话 可以建议大家去b站看一下这个视频
#b站视频链接(https://www.bilibili.com/video/BV1da4y1p7iZ)
#这个视频详细的介绍了正则表达式的使用 以及还推荐了一个很实用的在线的正则表达式检验的网站 同学们可以去学习一下
#正则表达式检验网站(https://regex101.com/)

content=re.findall(novel,result)  #调用re模块 re.finaall是用来查找result中与novel所匹配的内容 以列表的形式返回所匹配到的字符串
#将所匹配到的字符串存储在content列表中

with open('笔趣阁爬虫数据.txt','w',encoding='utf-8') as f:  #以写入的方式将所提取的数据存入txt文件中 'w'代表写入 encoding='utf-8'代表声明文件的编码方式
    for i in content:  #循环遍历列表
         s = i+'\n'    #'\n'代表换行符 每写入一个列表元素之后进行换行
         f.write(s)    #将列表元素写入文件中
