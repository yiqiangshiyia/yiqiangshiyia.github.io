数字类型转换

eval()和float()函数都可以把input() 函数接收到的数值型字符串转为数值类型

a = eval(input())   #输入整数字符串，转换为整数；输入浮点数字符串，转换为浮点数
b = float(input())  #输入整数和浮点数，都转换为浮点数
c = int(input())    #只接受整数输入，输出为整数

例：float(‘3.14’) 的结果是数值 3.14

int()函数可以把整数字符串转为整数

例：int(‘5’) 的结果是数值 5

格式化字符串

1.format格式化字符串

format（）方法接受位置参数和关键字参数，两者均传递一个叫做replacement字段，而这个replacement字段在字符串内由大括号表示。

位置参数：

字符串中的{0}{1}{2}表示位置，依次被format中的三个参数代换，format中的三个参数就叫做位置参数

str="{0} prety {1} {2}".format("I","love","python")
print(str)

I prety love python

关键字参数：format将参数中等值的字符串替换进去，这就是关键字参数

字符串中的{a}{b}{c}相当于三个标签，

str="{a} prety {b} {c}".format(a="I",b="love",c="python")
print(str)

I prety love python

位置参数和关键字参数同时用：

str="{a} prety {b} {c}".format(a="I",b="love",c="python")
print(str)

lambda函数学习

lambda函数是一种匿名函数，python允许使用lambda 关键字来创建匿名函数，匿名函数比普通函数要简洁。

lambda函数的使用方法：

python中lambda函数的使用非常的精简，基本语法是在冒号（：）左边放原函数的参数，可以有多个参数，用逗号（，）隔开即可：冒号右边是返回值。

举例：

当只有一个参数的时候：

普通函数：

def ds(x):
  return 2*x+1
y=ds(5)
print(x)

输出值为11

匿名函数：

y=lambda x : 2*x+1
print(y(5))

输出值为11

当有两个或者两个以上的参数时：

普通函数：

def add(x,y):
  return x+y
z=add(3,4)
print(z)

匿名函数：

z=lambda x,y : x+y
print(z(3,4))

lambda函数的作用：

1.python写一些执行脚本时，使用lambda就可以省下定义函数过程，比如说只是需要写个简单的脚本来管理服务器时，就不需要专门定义一个函数然后再写调用，使用lambda函数就可以使代码变得更加精简。

2．对于一些比较抽象并且整个程序执行下来只需要调用一两次的函数，有时候给函数起个名字也是比较头疼的问题，使用lambda函数就不需要 考虑命名的问题了。

3.简化代码的可读性，由于阅读普通函数经常要跳到开头def定义的位置，使用lambda函数就可以省去这样的步骤

列表

string1=input()
list_string=list(string1)  #将字符串转化成列表
print(list_string)
#向列表中添加元素
list_string.append()  #append()可以在列表末尾添加单个元素
list_string.extend([])  #extend()可以在列表末尾添加多个元素 且元素是以列表的形式添加的
list_string.insert(1,0)  #insert()可以在列表的任意位置添加元素 insert()有两个参数 第一个参数代表在列表中的插入位置 第二个参数代表在列表中插入的元素
#从列表中获取元素 类似于数组
#列表分片 列表的分片可以更加快速的获取列表中的元素
list_string[0:10]  #分号前后的两个参数值分别代表元素的位置
list_string[0:10:2]  #列表分片还可以引入第三个参数步长 步长代表每前进一个步长后再取出一个元素
list_string[::-1]  #特殊情况 当步长为-1时 代表输出一个反转列表
#从列表中删除元素
list_string.remove("")  #无需知道元素的位置 若不存在则会报错 remove不能删除指定位置的元素
del list_string[0]  #del是语句不是列表方法 不用小括号 []中元素代表删除元素的位置
list_string.pop()  #pop默认弹出列表中的最后一个元素 当在()中加上索引值也可以删除具体位置的列表元素
#列表比较大小 列表的+ *
list_string+list_string  #+表示连接两个列表
list_string*3  #*表示对列表进行3次复制
list_string.count(1)  #count()用于计算()中参数在列表中出现的次数
list_string.index(1)  #index()用于返回()中参数在列表中的位置
list_string.index(1,0,10)  #index()方法还有两个参数 限定查找范围
list_string.reverse()  #reverse()对列表进行反转
list_string.sort()  #sort()对列表进行排序
string2=''.join(list_string)  #将列表转化成字符串
print(string2)
